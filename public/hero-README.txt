Bandeau d’accueil — image optionnelle
=====================================
Par défaut : dégradé rose (sans fichier).

• Optionnel : déposez hero.png ou hero.jpg dans ce dossier (public/) pour une image de fond sur l’accueil.
• Ou : réglage dans Admin → Accueil → « Fond du bandeau d’accueil ».
• Variable .env : VITE_HERO_BG=https://… (prioritaire sur le reste).

Le fond général des pages est un dégradé léger (sans image florale).
